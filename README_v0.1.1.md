# Pledge Manager Starter v0.1.1

This revision fixes the GitHub Pages faceplant from v0.1.0 by adding a root `index.html` plus a tiny `config.js` file.

## What changed
- Added `index.html` so GitHub Pages has an entry point.
- Added `config.js` and `config.example.js` for Supabase settings.
- Added `assets/app.js` and `assets/styles.css` for a simple starter shell.
- Kept the original schema and planning docs in `/docs`.

## How to use
1. Run the SQL in `docs/supabase_schema_v0.1.0.sql`.
2. Edit `config.js` with your Supabase URL and anon key.
3. Push the folder contents to your GitHub Pages repo root.
4. Open the site. It should load instead of returning a 404.

## Important
This still is not the finished app. It is a GitHub Pages-ready starter package with the planning docs attached.
