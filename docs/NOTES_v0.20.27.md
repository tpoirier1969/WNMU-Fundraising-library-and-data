# WNMU Pledge Program Library v0.20.27

## This revision
- Added drag-and-drop report import area alongside the existing file picker.
- Improved unmatched-title manual match dropdowns so likely title matches rise to the top for each row.
- Tightened unmatched import table column sizing so the manual-match controls fit better on one screen.
- Added fundraiser scheduler report/import/difference money fields in the fundraiser header.
- Expanded fundraiser detail cards so each scheduled title shows dollars earned within the current fundraiser, airing count, and average per airing.
- Constrained the calendar grid to its own scrollable area and reinforced sticky day/time headers.

## Scheduler behavior reminder
- **Build from imported reports** merges imported rows into fundraiser schedules and skips duplicate placements.
- **Rebuild imported schedules** clears prior imported placements for those fundraisers, then recreates them from the imported rows.
